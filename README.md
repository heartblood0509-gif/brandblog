# 브랜드 블로그 자동 생성 도구

브랜드 블로그 글을 레퍼런스 형식에 맞춰 자동으로 생성하는 PyQt5 기반 GUI 애플리케이션입니다.

## 주요 기능

### 1. 레퍼런스 글 분석
- 텍스트 파일(.txt, .md) 불러오기 또는 직접 입력 지원
- Claude Sonnet 4.5를 사용한 글의 구조 자동 분석
  - 제목 패턴 및 스타일
  - 서론 구성 방식
  - 본론 섹션 구조
  - 소제목 스타일
  - 문단 길이와 구성
  - 결론 방식
  - 문체 특징 (문장 길이, 어조, 키워드 밀도)

### 2. 새 글 생성
- 새로운 주제와 키워드 입력
- 레퍼런스와 동일한 구조 및 문체로 글 생성
- 실시간 편집 가능한 텍스트 에디터

### 3. 편집 및 저장
- 생성된 글 실시간 수정
- 다양한 형식으로 저장
  - .txt (텍스트)
  - .md (마크다운)
  - .html (네이버 블로그용)

## 기술 스택

- **Python 3.14+**
- **PyQt5** - GUI 프레임워크
- **Google Gemini API** - Gemini 2.5 Flash 모델 (최신)
- **Playwright** - 강력한 웹 크롤링 (iframe, JavaScript 지원)
- **python-dotenv** - 환경 변수 관리

## 설치 방법

### 1. 저장소 클론 또는 다운로드

```bash
cd /path/to/test1
```

### 2. 가상환경 생성 및 활성화

```bash
# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화 (macOS/Linux)
source venv/bin/activate

# 가상환경 활성화 (Windows)
venv\Scripts\activate
```

### 3. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. API 키 설정

1. `.env.example` 파일을 `.env`로 복사:
```bash
cp .env.example .env
```

2. `.env` 파일을 열고 Google Gemini API 키 입력:
```
GEMINI_API_KEY=your_api_key_here
```

Google Gemini API 키는 [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)에서 **무료로** 발급받을 수 있습니다.

## 사용 방법

### 프로그램 실행

```bash
python blog_generator.py
```

### 사용 순서

1. **레퍼런스 글 준비**
   - 좌측 패널에서 "파일 불러오기" 버튼을 클릭하여 레퍼런스 글 파일을 불러오거나
   - 텍스트 박스에 직접 레퍼런스 글을 입력합니다

2. **구조 분석**
   - "구조 분석" 버튼을 클릭하여 레퍼런스 글의 구조와 문체를 분석합니다
   - 분석 결과가 하단에 표시됩니다

3. **새 글 생성**
   - 우측 패널에서 다음 정보를 입력합니다:
     - **주제**: 작성할 블로그 글의 주제
     - **타겟 키워드**: 포함할 키워드 (쉼표로 구분)
     - **추가 요구사항**: 선택적으로 추가 요구사항 입력
   - "글 생성" 버튼을 클릭합니다

4. **편집 및 저장**
   - 생성된 글을 확인하고 필요시 직접 수정합니다
   - 저장 형식을 선택하고 "저장" 버튼을 클릭합니다

## 주요 클래스 및 기능

### BlogGeneratorApp
메인 GUI 애플리케이션 클래스

**주요 메서드:**
- `init_ui()`: UI 초기화 및 다크모드 적용
- `load_reference_file()`: 레퍼런스 파일 불러오기
- `analyze_reference()`: Claude API를 사용한 구조 분석
- `generate_content()`: 새로운 블로그 글 생성
- `save_content()`: 생성된 글 저장

### AnalyzeThread
레퍼런스 분석을 백그라운드에서 처리하는 스레드
- UI 블록킹 방지
- 실시간 상태 업데이트

### GenerateThread
글 생성을 백그라운드에서 처리하는 스레드
- UI 블록킹 방지
- 실시간 상태 업데이트

## 주의사항

- Google Gemini API는 **무료 할당량**이 제공되며, 초과 시 비용이 발생할 수 있습니다
- API 키는 절대 공개하지 마세요
- `.env` 파일은 `.gitignore`에 추가하여 버전 관리에서 제외하세요

## 트러블슈팅

### API 키 오류
```
.env 파일에 GEMINI_API_KEY를 설정해주세요.
```
→ `.env` 파일이 올바르게 생성되었는지, API 키가 정확한지 확인하세요.

### 모듈을 찾을 수 없음
```
ModuleNotFoundError: No module named 'google.genai'
```
→ 가상환경이 활성화되었는지 확인하고 `pip install -r requirements.txt`를 실행하세요.

### Playwright 브라우저 미설치
```
playwright._impl._errors.Error: Executable doesn't exist
```
→ Playwright 브라우저를 설치하세요: `playwright install chromium`

### PyQt5 관련 오류 (macOS)
```
This application failed to start because no Qt platform plugin could be initialized.
```
→ PyQt5를 재설치하세요: `pip uninstall PyQt5 && pip install PyQt5`
